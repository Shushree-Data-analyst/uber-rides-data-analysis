# 🚀 Uber Rides Data Analysis

## 📌 Project Overview
This project analyzes Uber ride data using Python, SQL, and data visualization techniques to uncover patterns in ride behavior, usage trends, and customer preferences.

---

## 🎯 Objectives
- Analyze ride booking patterns
- Identify peak hours and usage trends
- Understand ride purposes and categories
- Explore distance distribution of trips

---

## 🧰 Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- SQL (MySQL)
- Jupyter Notebook

---

## 📂 Dataset
- Uber ride dataset containing:
  - Start Date & End Date
  - Category (Business/Personal)
  - Start & Stop Locations
  - Purpose
  - Distance (Miles)

---

## 🔧 Data Preprocessing
- Handled missing values in PURPOSE column
- Converted date columns to datetime format
- Extracted:
  - Date
  - Time
  - Day/Night categories
- Removed duplicates and null rows

---

## 📊 Key Analysis

### 🚖 Ride Categories
- Most rides are categorized as **Business trips**

### 🎯 Ride Purpose
- Major purposes:
  - Meetings
  - Meal / Entertainment

### ⏰ Time Analysis
- Peak ride time: **10 AM – 5 PM (Afternoon)**

### 📅 Monthly Trends
- Ride frequency is inconsistent across months
- Lower rides observed in winter months

### 📆 Weekly Trends
- Certain weekdays show higher ride demand

### 📏 Distance Insights
- Most rides are within **0–20 miles**
- Peak usage around **4–5 miles**
